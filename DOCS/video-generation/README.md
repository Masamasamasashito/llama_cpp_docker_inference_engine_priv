# 動画生成（ComfyUI）

RTX 5090 上でローカル動画生成AIを動かすためのガイドです。
Diffusionベースのモデルを ComfyUI 経由で実行します。

> テキスト生成LLM（llama.cpp）とは別の仕組みです。

---

## ドキュメント

- [セットアップ手順](01_setup_guide.md) - ComfyUIの導入から動画生成まで
- [利用可能モデル一覧](02_available_models.md) - 動画生成モデルの比較・VRAM要件

## 対応モデル

| モデル | 速度 | 品質 | VRAM目安 |
|---|---|---|---|
| LTX-Video | 最速（5秒動画を約4秒） | 良好 | ~8GB |
| Wan 2.1/2.2 (14B) | 4-15分/クリップ | 最高 | ~12-16GB |
| HunyuanVideo 1.5 | 4-15分/クリップ | 高（人物得意） | ~16GB |

RTX 5090（32GB VRAM）ではすべてのモデルが余裕で動作します。
