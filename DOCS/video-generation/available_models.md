# 動画生成モデル一覧

RTX 5090（VRAM 32GB）で動作する動画生成モデルの比較です。
すべてComfyUI上で実行します。

---

## LTX-Video

| 項目 | 内容 |
|---|---|
| 開発元 | Lightricks |
| モデルサイズ | ~2B |
| VRAM目安 | ~8GB |
| 生成速度 | 5秒動画を約4秒（RTX 5090） |
| 特徴 | 最速。リアルタイムに近い動画生成 |
| 対応解像度 | 512x512, 768x512 |
| ライセンス | Apache 2.0 |

NVIDIA公式がRTX向けに最適化を推進しており、ComfyUIとの統合も進んでいます。

---

## Wan 2.1 / 2.2

| 項目 | 内容 |
|---|---|
| 開発元 | Alibaba / Wan Team |
| モデルサイズ | 1.3B / 14B |
| VRAM目安 | ~8GB（1.3B） / ~12-16GB（14B） |
| 生成速度 | 4-15分/クリップ（14B） |
| 特徴 | 最高品質。テキストから高精細動画生成 |
| 対応解像度 | 最大720p |
| ライセンス | Apache 2.0 |

14Bモデルが最高品質。1.3Bは軽量で8GB GPUでも動作します。

---

## HunyuanVideo 1.5

| 項目 | 内容 |
|---|---|
| 開発元 | Tencent |
| モデルサイズ | ~13B |
| VRAM目安 | ~16GB |
| 生成速度 | 4-15分/クリップ |
| 特徴 | 人物の顔が得意。VRAM使用量40%削減済み（v1.5） |
| 対応解像度 | 最大720p |
| ライセンス | Tencent Hunyuan License |

v1.5でVRAM使用量が大幅に削減され、RTX 5090なら余裕で動作します。
ComfyUIのカスタムノードで利用可能。

---

## モデル比較まとめ

| モデル | 速度 | 品質 | 人物 | VRAM | おすすめ用途 |
|---|---|---|---|---|---|
| **LTX-Video** | 最速 | 良好 | 普通 | ~8GB | プロトタイプ・リアルタイム生成 |
| **Wan 2.2 (14B)** | 遅い | 最高 | 良好 | ~16GB | 高品質動画制作 |
| **Wan 2.1 (1.3B)** | 中程度 | 良好 | 普通 | ~8GB | 軽量環境・テスト |
| **HunyuanVideo 1.5** | 遅い | 高い | 最高 | ~16GB | 人物中心の動画 |
