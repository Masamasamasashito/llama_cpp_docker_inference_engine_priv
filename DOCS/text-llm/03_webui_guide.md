# 3. WebUI ガイド

llama.cppサーバーには**組み込みWebUI**が搭載されています。
追加インストール不要で、ブラウザからChatGPT風のチャット画面が利用できます。

> WebUIはLLMサーバーPC（Windows 11 Pro）のブラウザからのみ利用します。

---

## アクセス方法

LDIEサーバー起動後、Windows 11 Proのブラウザで以下にアクセス:

```
http://localhost:8081
```

| モード | URL |
|---|---|
| GPU版（デフォルト） | `http://localhost:8081` |
| CPU版 | `http://localhost:8082` |
| High版（RTX 5090） | `http://localhost:8083` |

> ポートを変更している場合は `.env` の `DOCKER_HOST_PORT_LLAMA` の値に合わせてください。

---

## 主な機能

| 機能 | 説明 |
|---|---|
| チャット | ChatGPT風のマルチターン会話 |
| テキストファイル添付 | テキストファイルをコンテキストに追加できる |
| PDF添付 | PDFを添付して内容について質問可能（テキスト抽出またはモデルが対応していれば画像として処理） |
| パラメータ調整 | Temperature、Top-P、Top-K等をUI上で変更可能 |
| システムプロンプト | カスタムシステムプロンプトの設定 |
| 会話履歴 | セッション内の会話履歴を保持 |

---

## API Key認証が有効な場合

`LLAMA_API_KEY` を設定してサーバーを起動した場合、WebUIからの利用にも認証が必要です。

WebUIにアクセスすると認証プロンプトが表示されます。
`.env` に設定した `LLAMA_API_KEY` の値を入力してください。

> `LLAMA_API_KEY` が空（未設定）の場合は認証なしでアクセスできます。
> ローカルのみ（`DOCKER_HOST_BIND_ADDR=127.0.0.1`）の場合は認証なしでも安全です。

---

## WebUI vs API の使い分け

| 用途 | WebUI | API（curl / Python） |
|---|---|---|
| 手動での対話・質問 | 向いている | 不向き |
| ファイルを見せて質問 | 向いている（ドラッグ&ドロップ） | コード記述が必要 |
| パラメータの実験 | 向いている（UIでスライダー調整） | .envまたはコード変更 |
| 自動化・バッチ処理 | 不向き | 向いている |
| OpenClaw連携 | 不可 | API経由 |
| プログラムからの呼び出し | 不可 | 向いている |

## トラブルシューティング

| 問題 | 原因・対策 |
|---|---|
| ページが開かない | LDIEサーバーが起動しているか `docker ps` で確認 |
| 「Connection refused」 | ポート番号が合っているか確認（GPU: 8081, CPU: 8082, High: 8083） |
| 入力しても応答がない | モデルのロードに時間がかかっている可能性。`docker logs` でステータス確認 |
| 認証エラー | `LLAMA_API_KEY` の値を正しく入力しているか確認 |
